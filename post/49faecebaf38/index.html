学习python的笔记 此处Python版本是Python 3.8

安装Python环境

参考用 Miniconda 和 Geany 轻松打造 Python 开发环境 - 知乎 (zhihu.com)

使用Conda安装各种包

Miniconda3 可以通过 pip 和 conda 两个命令模式来下载、安装所需要的包，但如上所述，最好用 conda 命令来进行；只有当利用 conda 命令确实无法下载、安装的时候，才使用 pip 命令。

在安装了Miniconda并配置了国内镜像^1的情况下可以直接打开命令行敲入

命令行模式与Python交互模式

打开cmd(win+r -> input cmd -> enter)

输入python发现提示符变成>>>即进入python交互模式，在交互模式下输入exit()并回车即退出python交互模式

交互模式下直接输入代码按回车就可以得出运行结果
注意命令行模式下可以执行python进入python交互式环境，也可以通过python filename.py运行一个py文件，但是交互模式不可以执行.py文件

在包含py文件的目录路径输入cmd可快速进入改路径

也可以使用cd命令切换路径Windows命令行cmd之cd命令用法_张三的博客-CSDN博客_windows命令行cd

常用cd命令

  • cd/ 回到根目录
  • cd..回到上一层目录
  • cd /d [对应目录] 进入到[对应目录]中
  • cd/?显示cd帮助及用法

发布

打包成exe

打包python文件为exe程序 vscode - 菜鸟要飞~ - 博客园 (cnblogs.com)

下载pyinstaller

在需要打包的文件目录下输入打包/在vscode终端输入貌似也可以

pyinstaller -F setup.py #打包exe

pyinstaller -F -w setup.py #不带控制台的打包

pyinstaller -F -i ab.ico setup.py #打包指定图标的exe

pyinstaller -F -w -i abc.ico setup.py

-F 代表之作独立的可执行程序

-w 程序启动时不会打开命令行 （如果程序中有print函数，就不能加w，否则会报错)

-i 设定自己的图标，也可以写成 –icon=abc.ico
